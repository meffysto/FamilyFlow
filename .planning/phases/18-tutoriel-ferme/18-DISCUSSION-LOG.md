**Clarification Claude** : Puisque D-01 a décidé que les étapes 1 et 5 sont des cartes narratives pleine largeur (pas spotlights), la vraie question devient : **quel contenu visuel décorer** ces cartes ?

### Area 4 — Forme du spotlight

**Question** : Rectangle arrondi (extension simple de `CoachMarkOverlay`) ou cercle SVG `<Mask>` (réécriture avec react-native-svg) ?

**Options présentées** :
- (A) **Rectangle arrondi** — le plus simple, étend l'existant
- (B) Cercle SVG — plus joli pour cibles rondes, ajoute dépendance SVG mask

**Réponse user** : "le plus simple"

**Décision** :
- D-05 = Rectangle arrondi, extension de `CoachMarkOverlay` avec nouveau prop `borderRadius`
- D-05bis = SVG Mask explicitement rejeté

### Area 5 — Pause des animations WorldGridView

**Question** : Comment pauser les animations ? (a) state HelpContext + useMemo dans WorldGridView, (b) prop `paused` explicite, (c) flag global dans nouveau module.

**Options présentées** :
- (a) State HelpContext consumé par WorldGridView — DRY mais couple HelpContext et WorldGridView
- (b) Prop `paused` explicite — propre, pas de couplage, facile à tester
- (c) Flag global nouveau module — inutilement indirect

**Réponse user** : "je te fais confiance"

**Décision Claude (par délégation)** :
- D-06 = Prop `paused` explicite sur `WorldGridView`
- D-06bis = Source de vérité = état `activeFarmTutorialStep` dans HelpContext, lu par `tree.tsx` qui le passe au WorldGridView. Meilleur des deux : pas de prop drilling multi-niveaux + composant découplé côté WorldGridView.
