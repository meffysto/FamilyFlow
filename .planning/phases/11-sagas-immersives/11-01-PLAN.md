  <action>
    Creer `components/mascot/VisitorSlot.tsx` — composant React Native (pas SVG) en position absolute dans le diorama, calque sur le pattern AnimatedAnimal/CompanionSlot (per D-04).

    **Interface :**
    ```typescript
    export interface VisitorSlotProps {
      visible: boolean;             // true = chapitre disponible aujourd'hui
      sagaId: string;               // pour tint variation par saga
      containerWidth: number;
      containerHeight: number;
      onTap: () => void;            // declenchera SagaWorldEvent dans tree.tsx
      shouldDepart: boolean;        // true apres completion chapitre
      isLastChapter: boolean;       // true = depart dramatique (flash + etoiles)
      onDepartComplete?: () => void; // callback apres animation depart finie
    }
    ```

    **Etats internes (per D-08, D-09, D-10, D-11) :**
    ```typescript
    type VisitorState = 'entering' | 'idle' | 'departing' | 'departed';
    ```

    **Position dans le diorama (Claude's Discretion, suggestion recherche) :**
    - Position cible : `fx: 0.72, fy: 0.62` (cote droit, legerement plus bas que le chemin principal pour eviter collision avec CompanionSlot PATROL_ROUTE)
    - Point de depart arrivee : `fx: 1.15` (hors bord droit)
    - Point de sortie depart : `fx: 1.20` (hors bord droit)

    **VISITOR_SIZE = 48** (meme taille que les sprites generes)

    **Animation arrivee (per D-08) :**
    - `const posX = useSharedValue(containerWidth * 1.15)` — commence hors ecran droite
    - `posX.value = withSpring(containerWidth * 0.72, { damping: 16, stiffness: 120 })` — marche vers position cible
    - Frame swap walk_left (6 frames, 150ms/frame via setInterval + useState(frameIdx)) pendant la marche
    - `scaleX: 1` sur Image (personnage marche vers la gauche = bonne direction depuis droite)
    - `opacity.value = withTiming(1, { duration: 200 })` au debut
    - Quand spring termine → `runOnJS(setState)('idle')`

    **Animation idle (per D-09) :**
    - `bounceY = useSharedValue(0)` — `withRepeat(withSequence(withTiming(-3, 600ms), withTiming(0, 600ms)), -1, true)`
    - Frame swap idle (2 frames, 800ms/frame via setInterval)
    - Bulle "!" pulsante : `Animated.View` en `position: 'absolute', top: -18, alignSelf: 'center'`
      - `bubbleScale = withRepeat(withSequence(withSpring(1.25, {damping:8,stiffness:200}), withSpring(1.0, {damping:12,stiffness:180})), -1, true)`
      - Contenu : `<Text>!</Text>` dans un cercle `backgroundColor: primary + 'DD'`, `borderRadius: Radius.full`, taille 16px
    - `Pressable onPress={onTap}` actif uniquement en etat 'idle'
    - `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light)` sur tap

    **Animation depart normal (per D-10) :**
    - Declenchee quand `shouldDepart` passe a true ET `state === 'idle'`
    - `posX.value = withSpring(containerWidth * 1.20, SPRING_WALK)` — marche vers la droite
    - Frame swap walk_left + `scaleX: -1` sur Image pour walk_right (pattern Phase 05-03)
    - Quand spring termine → `runOnJS(setState)('departed')` → appelle `onDepartComplete?.()`

    **Animation depart dramatique (dernier chapitre, per D-10) :**
    - Flash blanc : `flashOpacity = withSequence(withTiming(1, 200ms), withTiming(0, 600ms))`
    - Scale up + fadeout : `visitorScale = withSequence(withSpring(1.4, {damping:8,stiffness:120}), withTiming(0, 400ms))`
    - Apres animation → `runOnJS(setState)('departed')` → appelle `onDepartComplete?.()`

    **Variations saga (per D-01) :**
    - Prop `sagaId` utilisee pour determiner un `tintColor` optionnel sur l'Image
    - Map : `{ 'silver-voyager': '#C0C8D0', 'hidden-spring': '#80B4D0', ... }`
    - Si `tintColor` ne rend pas bien, utiliser un `<View>` overlay avec `opacity: 0.15` et `backgroundColor` (fallback mentionne dans la recherche)

    **Retour :** `if (state === 'departed') return null;`

    **Container Animated.View :**
    - `position: 'absolute'`, `left: TARGET_X`, `top: TARGET_Y`, `width: VISITOR_SIZE`, `height: VISITOR_SIZE`
    - `overflow: 'visible'` (pour que la bulle "!" depasse au-dessus)
    - Style anime : `transform: [{ translateX: posX.value - TARGET_X }, { translateY: bounceY.value }]` + `opacity`
