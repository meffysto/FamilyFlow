**Props interface:**
```typescript
interface CraftSheetProps {
  visible: boolean;

  coins: number;
```

**3 tabs** (horizontal Chip-style toggle, same pattern as TreeShop tabs):
- "Recettes" (catalogue) — default tab
- "Inventaire" (recoltes brutes)
- "Mes creations" (items craftes)

**Tab 1: Recettes** (per D-05):
- List of CRAFT_RECIPES as cards
- Each card shows: emoji, name (translated via labelKey), ingredients list with quantity and current stock
- Each ingredient shows: icon + name + "{{have}}/{{need}}" with green if sufficient, red if not
- "Crafter" button active only if `canCraft()` returns true
- On tap "Crafter":
  1. Call `onCraft(recipe.id)`
  2. On success: `Haptics.impactAsync(ImpactFeedbackStyle.Medium)` + show toast via `useToast()` with craftReussi message
  3. Use `withSpring` scale animation on the crafted item (scale 0 -> 1.2 -> 1)

**Tab 2: Inventaire** (recoltes brutes):
- List of crops in harvestInventory where quantity > 0
- Each row: crop emoji (from CROP_CATALOG) + name + quantity + sell value
- "Vendre" button on each row
- On tap "Vendre": call `onSellHarvest(cropId)`, show toast "+N 🍃", haptic

**Tab 3: Mes creations** (per D-07, D-08, D-09):
- List of craftedItems grouped by recipeId
- Each row: recipe emoji + name + count + sell value (recipe.sellValue)
- "Vendre" button on each row
- On tap "Vendre": call `onSellCrafted(recipeId)`, show toast "+N 🍃", haptic

**Styling:**
- Use `useThemeColors()` for all colors — NO hardcoded values
- Use Spacing, Radius, FontSize, FontWeight, Shadows tokens
- Cards with `Shadows.sm`, `Radius.lg` border radius
- Ingredient availability: `colors.success` for sufficient, `colors.error` for insufficient
- StyleSheet.create at bottom for static styles
- Dynamic styles inline with theme colors
