**1. Modify `lib/mascot/farm-engine.ts` — Change harvestCrop return type:**

Current signature returns `{ crops, reward }`. Change to return `{ crops, harvestedCropId, isGolden }` so the caller decides what to do with the harvest (add to inventory instead of giving coins).

```typescript
export function harvestCrop(
  crops: PlantedCrop[],
  plotIndex: number,
): { crops: PlantedCrop[]; harvestedCropId: string | null; isGolden: boolean } {
  const crop = crops.find(c => c.plotIndex === plotIndex);
  if (!crop || crop.currentStage < MAX_CROP_STAGE) {
    return { crops, harvestedCropId: null, isGolden: false };
  }
  return {
    crops: crops.filter(c => c.plotIndex !== plotIndex),
    harvestedCropId: crop.cropId,
    isGolden: crop.isGolden ?? false,
  };
}
```

Remove the `reward` calculation from harvestCrop entirely — it is now the responsibility of useFarm to add to inventory.
