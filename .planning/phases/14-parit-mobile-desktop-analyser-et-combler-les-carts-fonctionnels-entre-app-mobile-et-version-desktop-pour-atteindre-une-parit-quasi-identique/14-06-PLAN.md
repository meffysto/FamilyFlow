**Loot.tsx — Structure en 3 sections :**

**Section 1 — Coffre a ouvrir (haut de page) :**
- Afficher le nombre de coffres disponibles (`activeProfile.lootBoxesAvailable`)
- Bouton "Ouvrir un coffre" (desactive si 0 coffre)
- Click declenche `openLootBox()` depuis useVault
- Animation d'ouverture en 3 phases :
  1. Coffre tremble (Framer Motion `animate={{ x: [-2, 2, -2, 2, 0] }}` sur 0.5s)
  2. Card flip reveal (per D-03 et RESEARCH Pattern 4) :
     ```typescript
     import { motion, AnimatePresence } from 'framer-motion';
     // Parent : style={{ perspective: 1200 }}
     // Card : initial={{ rotateY: 180, scale: 0.85, opacity: 0 }}
     //        animate={{ rotateY: 0, scale: 1, opacity: 1 }}
     //        transition={{ type: 'spring', stiffness: 160, damping: 18 }}
     ```
  3. Glow par rarete (per RESEARCH) :
     ```typescript
     const RARITY_GLOW: Record<string, string> = {
       commun: '0 0 8px rgba(100,100,100,0.4)',
       rare: '0 0 16px rgba(59,130,246,0.6)',
       epique: '0 0 24px rgba(168,85,247,0.8)',
       legendaire: '0 0 32px rgba(245,158,11,1.0)',
     };
     ```
  4. Confetti :
     ```typescript
     import confetti from 'canvas-confetti';
     // Apres le reveal :
     confetti({ particleCount: 100, spread: 70, origin: { y: 0.6 } });
     // Pour legendaire : particleCount: 200, spread: 100
     ```
- Card affiche : icone reward, nom, rarete (badge couleur), description

**Section 2 — Inventaire des rewards (milieu) :**
- SegmentedControl : "Tous", "Decorations", "Habitants", "Compagnons", "Cosmétiques"
- Grille de cards pour chaque reward obtenu
- Chaque card : icone, nom, rarete badge, bouton "Utiliser" si applicable
- Hover effet scale (CSS transition)

**Section 3 — Collection badges (bas) :**
- Grille de badges/achievements
- Badges obtenus : colores, avec date d'obtention
- Badges non obtenus : grisés, avec condition de deblocage
- Barre de progression collection (ex: "15/30 badges")

**CSS :**

- Grille inventaire : auto-fill, minmax(140px, 1fr)
- Badges : cercles avec bordure rarete
- Glow animations : `@keyframes glow` ou box-shadow transition
