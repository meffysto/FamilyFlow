**Types :**
```typescript
export interface GiftEntry {
  sender_id: string;
  sender_name: string;
  sender_avatar: string;
  item_type: 'harvest' | 'rare_seed' | 'crafted' | 'building_resource';
  item_id: string;
  quantity: number;
  sent_at: string;
}

export interface PendingGifts {
  gifts: GiftEntry[];
}

export interface GiftHistoryEntry {
  date: string;
  direction: 'sent' | 'received';
  fromId: string;
  toId: string;
  itemType: string;
  itemId: string;
  quantity: number;
}

export const MAX_GIFTS_PER_DAY = 5;
```

**Fonctions :**
- `parsePendingGifts(content: string): PendingGifts` — gray-matter, defensive `?? []` sur `data.gifts`
- `serializePendingGifts(gifts: GiftEntry[]): string` — `matter.stringify('', { gifts })`
- `canSendGiftToday(giftsSentField: string | undefined, now?: Date): boolean` — split `|`, compare date, count < 5
- `incrementGiftsSent(giftsSentField: string | undefined, now?: Date): string` — format `count|YYYY-MM-DD`
- `addGiftToInventory(farmData: FarmProfileData, entry: GiftEntry): FarmProfileData` — switch sur item_type, retourne copie modifiee. Pour harvest: `{ ...inv, [itemId]: (inv[itemId] ?? 0) + qty }`. - `buildGiftHistoryEntry(direction: 'sent' | 'received', fromId: string, toId: string, itemType: string, itemId: string, qty: number, now?: Date): string` — format `ISO|direction|fromId->toId|type:itemId:qty`
