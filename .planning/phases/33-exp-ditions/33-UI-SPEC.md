## Layout Contract

### Modal: ExpeditionsSheet

```
Modal (pageSheet, presentationStyle)
├── AwningStripes (Farm chrome header)
├── ModalHeader ("Expéditions ⚔️", drag-to-dismiss)
├── Tab row (3 tabs: Catalogue / En cours / Résultats)
│   └── Tab highlight: primary color underline (8px horizontal padding each side)
├── ScrollView (per active tab)
│   ├── [Catalogue tab] Pool du jour — 3 ExpeditionCards (Facile / Moyen / Dur)
│   ├── [En cours tab] Up to 2 ActiveExpeditionRows
│   └── [Résultats tab] Resolved expeditions, newest first
└── Bottom safe area padding
```

- Modal presented via `presentationStyle="pageSheet"` + drag-to-dismiss (matches D-12)
- Spring animation on open: `{ damping: 12, stiffness: 180 }` (matches SPRING_CATALOG in BuildingsCatalog)
- Tab switching: instant (no cross-fade), underline indicator animates with withTiming 150ms

### Card: ExpeditionCard

```
TouchableOpacity (44px min height row, Radius.lg border)
├── Row left: destination emoji (FontSize.icon = 28) + info column
│   ├── Expedition name (FontSize.subtitle, semibold)
│   └── Duration text (FontSize.label, textMuted)
├── Difficulty badge (colored bg chip, FontSize.label, semibold)
├── Cost row: feuilles chip + récolte chip
└── [Si active slot disponible] "Lancer" button (primary bg, 44px height)
    [Si 2 actives] disabled state ("Complet", textMuted, opacity 0.5)
```

### Component: CampExplorationCell

```
View (slot dans grille ferme, same size as other farm cells)
├── Image (camp sprite, pixel art)
├── [Si expeditions actives] Badge overlay top-right
│   └── Text "1/2" ou "2/2" (FontSize.micro, Farm.gold bg, 8px horizontal padding)
└── [Si expédition terminée] Badge overlay bottom
    └── Text "Retour !" (FontSize.micro, colors.success bg, 8px horizontal padding, pulse animation)
```

### Screen: Résultat (ExpeditionChest)

```
Modal/overlay (fullscreen, overlay background)
├── Chest sprite (centered, 96x96 initial, 128x128 post-open)
├── [Avant tap] Text "Appuie pour ouvrir !" (FontSize.label, textMuted)
├── [Après tap] Reveal row: récompense emoji + nom + type badge
└── [Bouton] "Fermer le coffre" (secondary style, 44px)
```

## Interaction Contract

### Launch expedition (VILL-16)

1. User taps "Lancer" on ExpeditionCard
2. Alert.alert() confirmation — title "Confirmer l'expédition ?", message includes exact cost + outcome probabilities + "La mise est perdue en cas d'échec." (D-07 — risque réel communiqué)
3. On confirm: Haptics.impactAsync(ImpactFeedbackStyle.Medium) + deduct resources + write vault + update active count badge on CampExplorationCell
4. On cancel: no-op

### View active timer (VILL-17)

- CampExplorationCell shows badge "1/2" or "2/2" always when expeditions active
- Tapping camp opens ExpeditionsSheet at "En cours" tab
- Each ActiveExpeditionRow shows countdown formatted as "Hh Mm" (date-fns or manual)
- When < 30 min remaining: countdown text switches to colors.warning
- When complete: countdown replaced by "Collecter" button (primary style)

### Collect result (VILL-18)

1. User taps "Collecter" on ActiveExpeditionRow
2. Haptics.selectionAsync()
3. ExpeditionChest overlay appears (fade in, withTiming 200ms)
4. Chest is closed state — user sees "Appuie pour ouvrir !"
5. User taps chest: withSpring scale 1.0 → 1.3 → 1.0 + Haptics.notificationAsync(NotificationFeedbackType.Success)
6. Chest opens: reveal animation (opacity fade-in of reward, 300ms)
7. Result badge: colored per outcome (success/partial/failure/rare)
8. "Fermer le coffre" closes overlay, removes expedition from active list, appends to results history

### Failure state

- Result type "Échec": chest reveal shows skull emoji + "Expédition échouée" + "Ta mise a été perdue." (colors.error text)
- No haptic success — use Haptics.notificationAsync(NotificationFeedbackType.Error)
- No reward displayed

### Rare discovery

- Result type "Découverte rare": chest reveal uses Farm.gold text on Farm.parchment card + star particle burst (reuse AmbientParticles or simple opacity-fade stars)
- Haptics.notificationAsync(NotificationFeedbackType.Success) + 200ms delay + second Haptics.impactAsync(Heavy)

## Animation Contract

All animations use react-native-reanimated (CLAUDE.md mandatory). No RN Animated.

| Interaction | Animation | Config |
|------------|-----------|--------|
| Modal open | Reanimated spring scale 0.95 → 1.0 | `{ damping: 12, stiffness: 180 }` (SPRING_CATALOG) |
| "Lancer" button press | withSpring scale 1.0 → 0.96 → 1.0 | `{ damping: 10, stiffness: 200 }` |
| Chest closed → open | withSpring scale 1.0 → 1.3 → 1.0 | `{ damping: 8, stiffness: 160 }` |
| Reward reveal | withTiming opacity 0 → 1 | duration: 300ms, easing: Easing.out(Easing.ease) |
| Rare discovery stars | 4–6 star Views, withTiming opacity 1→0 + translateY 0→-40 | staggered delays 0/80/160/240ms |
| Countdown badge pulse (when complete) | withRepeat(withSequence(scale 1→1.08→1)) | 2 repeats, then stop |
| Tab indicator | withTiming translateX | duration: 150ms |

Spring config declared as module-level constant `const SPRING_CHEST = { damping: 8, stiffness: 160 } as const`.

Source: CONTEXT.md D-12/D-13, CLAUDE.md animation rules, BuildingsCatalog.tsx SPRING_CATALOG pattern
