Modifier `AmbientParticles.tsx` pour remplacer le `useMemo(() => getTimeSlot(), [])` statique par un systeme reactif :

1. **Remplacer le slot statique par un state reactif:**
   - `const [timeSlot, setTimeSlot] = useState(() => getTimeSlot());`
   - Supprimer la ligne `const timeSlot = useMemo(() => getTimeSlot(), []);`

2. **Ajouter un `useEffect` de polling toutes les 60 secondes:**
   ```typescript
   useEffect(() => {
     const interval = setInterval(() => {
       const newSlot = getTimeSlot();
       setTimeSlot(prev => prev === newSlot ? prev : newSlot);
     }, 60_000);
     return () => clearInterval(interval);
   }, []);
   ```

3. **Ajouter un overlay anime avec reanimated `useSharedValue` + `withTiming`:**
   - Importer `useSharedValue`, `useAnimatedStyle`, `withTiming` de `react-native-reanimated` et `Animated` de `react-native-reanimated`
   - Creer un `overlayOpacity = useSharedValue(0)` et `overlayR/G/B` shared values
   - Parser les valeurs rgba de `config?.colorOverlay` via une fonction helper `parseRgba(colorString: string): { r: number; g: number; b: number; a: number }`
   - Quand `timeSlot` change, animer vers la nouvelle opacite/couleur : `overlayOpacity.value = withTiming(targetA, { duration: 2000 });`
   - Pour le slot `jour` (config === null), animer `overlayOpacity` vers 0

4. **Remplacer le `<View>` overlay statique par un `<Animated.View>` anime:**
   - Remplacer `{config.colorOverlay && (<View style={[absoluteFill, { backgroundColor: config.colorOverlay }]} />)}`
   - Par un `<Animated.View style={[absoluteFill, overlayAnimStyle]} />` qui est TOUJOURS rendu (pas conditionnel) — l'opacite anime a 0 le cache quand c'est `jour`
   - Le `overlayAnimStyle` utilise `useAnimatedStyle` avec les shared values R, G, B, A pour generer `backgroundColor: rgba(...)`

5. **NE PAS modifier le rendu des particules** — elles continuent a fonctionner comme avant. Le changement de `config` via le state `timeSlot` fera que les particules apparaitront/disparaitront selon le slot (le `if (config === null) return null` doit etre ajuste : retourner uniquement l'overlay anime quand config est null, pas null complet).

6. **Gerer le cas `config === null` (jour):**
   - Ne plus retourner `null` quand `config === null`
   - Toujours rendre le composant avec l'overlay anime
   - Les particules ne se rendent que si `config !== null`
   - L'overlay anime vers opacite 0 pour le slot `jour`

7. **Respecter `useReducedMotion`:**
   - Si `reducedMotion === true`, appliquer directement la couleur sans `withTiming` (transition instantanee)
   - Pattern : `overlayOpacity.value = reducedMotion ? targetA : withTiming(targetA, { duration: 2000 });`

8. **Ajouter import de `useState` et `Animated` de reanimated si pas deja present.**
