Éléments ferme réutilisés
  • Farm.parchment / parchmentDark → fond des cartes
  • Farm.woodHighlight / woodLight → bordures et ombres basses
  • AwningStripes (9 rayures green+cream) → header du hero A
  • Farm.greenBtn / greenBtnShadow → bouton primaire + dock actif
  • Farm.woodBtn / woodBtnShadow → boutons 3D
  • Farm.orange / orangeShadow → pill de niveau et badges
  • Farm.progressGold / progressBg → barre XP (gradient doré)
  • text-shadow crème sur texte brun (comme les titres ExpeditionsSheet)
Quelle variante ?
  • A — cohérence max avec Atelier/Expéditions (auvent = signature ferme). Actions restent lisibles, hero ouvre la page comme un vrai panneau de ferme.
  • B — plus discret, garde le côté « dashboard ». Pas d'auvent (parfois trop visuel).
  • C — le plus game-y, panneau unique + hero doré. Gros contraste visuel mais moins de "breathing room".
Dis-moi ta préférée (ou un mix), je pars sur l'implé.
