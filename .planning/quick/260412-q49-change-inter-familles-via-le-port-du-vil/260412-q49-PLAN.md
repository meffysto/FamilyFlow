Creer lib/village/trade-engine.ts — module pur TypeScript, zero import hook/context (pattern identique a atelier-engine.ts).

**Types:**
```typescript
type TradeCategory = 'village' | 'farm' | 'harvest';
interface TradePayload {
  category: TradeCategory;
  itemId: string;
  quantity: number;
  timestamp: number; // epoch seconds
  nonce: number; // random 0-9999
}
```

**Encodage code-cadeau (format court et partageable):**
- Format: `FF-{category}{itemId}{qty}{timestamp}{nonce}{checksum}` 
- Encoder en base36 compact : concatener category (1 char: V/F/H), itemId (tronque 8 chars), quantity (2 digits), timestamp epoch en base36 (6 chars), nonce base36 (3 chars), checksum (2 chars).
- Le checksum = somme des code-points du payload modulo 1296 encode en base36 (2 chars).
- Prefixe `FF-` pour identifier les codes FamilyFlow.
- `encodeTrade(payload: TradePayload): string`
- `decodeTrade(code: string): TradePayload | null` — retourne null si format invalide ou checksum incorrect.

**Validation:**
- `isTradeExpired(payload: TradePayload, now?: Date): boolean` — true si > 48h depuis timestamp.
- `isTradeAlreadyClaimed(code: string, claimedCodes: string[]): boolean` — lookup dans la liste locale.
- `MAX_TRADES_PER_DAY = 5`

**Mapping itemId -> inventaire cible:**
- category 'village' → VillageInventory (itemIds: eau_fraiche, pain_frais, panier_surprise, cafe_matin, outil_forge, farine_moulee, coffre_maritime, parchemin)
- category 'farm' → FarmInventory (itemIds: oeuf, lait, farine, miel)
- category 'harvest' → HarvestInventory (itemIds: tous les cropIds du CROP_CATALOG)

**Helpers inventaire:**
- `getAvailableTradeItems(category: TradeCategory, villageInv: VillageInventory, farmInv: FarmInventory, harvestInv: HarvestInventory): { itemId: string; label: string; emoji: string; available: number }[]`
  - Pour village: iterer BUILDINGS_CATALOG.map(b => b.production) et retourner ceux avec qty > 0
  - Pour farm: iterer les 4 cles FarmInventory avec qty > 0
  - Pour harvest: iterer harvestInventory avec qty > 0, lookup emoji dans CROP_CATALOG

**Stockage codes reclames:** Liste de strings stockee dans gami-{id}.md via un nouveau champ `trade_claimed_codes?: string[]` dans FarmProfileData (format CSV dans le fichier markdown).

Ajouter `export * from './trade-engine';` dans lib/village/index.ts (barrel).
