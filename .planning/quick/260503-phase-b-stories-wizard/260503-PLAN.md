# Objectif

Réduire le wizard quotidien de génération d'histoires de 5-7 décisions à 3-4 en déplaçant les préférences durables (voix, multi-voix, mode audio, langue, longueur par défaut) vers un écran Settings par profil enfant.

# Scope

- Type `StoryDefaults` optionnel sur `Profile` (lib/types.ts).
- Bump `CACHE_VERSION` (Profile est cached).
- Helper `updateStoryDefaults` dans `hooks/useVaultProfiles.ts`.
- Nouvel écran `app/story-settings.tsx` (route hors-tabs, sélecteur enfant en haut, sections Voix / Mode audio / Multi-voix / Longueur / Langue).
- Bouton ⚙️ dans le header de `app/(tabs)/stories.tsx` à côté du 📖 → `router.push('/story-settings')`.
- Refonte `PersonnaliserStep` : retrait des pickers voix / multi-voix / spectacle / longueur ; conservation thème + chip override longueur (préselect = settings enfant).
- À la génération : merge `enfant.storyDefaults` + override → voiceConfig final.
- Pas de migration des histoires existantes.

# Décisions

- Settings stockés par enfant (champs dans frontmatter Profile).
- Override longueur reste visible dans le wizard (chip 1-tap), pas planqué dans "Avancé".
- Aucune migration : settings n'affectent que les nouvelles générations.

# Garde-fous

- `useThemeColors()` partout, zéro hardcode.
- Modals `pageSheet` + drag-to-dismiss.
- Animations via `react-native-reanimated`.
- Commits FR.
- `npx tsc --noEmit` avant chaque commit (ignorer erreurs pré-existantes useVault.ts / MemoryEditor.tsx / cooklang.ts).

# Plan

1. lib/types.ts — ajouter `StoryDefaults` + champ `storyDefaults?` sur Profile.

5. app/(tabs)/stories.tsx — bouton ⚙️ header + refonte PersonnaliserStep + merge défauts/override à la génération.
6. tsc + commits atomiques + SUMMARY.

# Validation

- `npx tsc --noEmit` propre (modulo erreurs pré-existantes connues).
- Wizard quotidien réduit à : enfant + univers + thème (+ override longueur visible).
- Settings persistent dans le frontmatter du profil enfant et sont relus au mount.
- Histoires existantes inchangées (leur voiceConfig sauvée reste tel quel).
