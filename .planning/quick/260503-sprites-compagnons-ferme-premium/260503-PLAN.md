# Objectif

Generer un premier pack de compagnons ferme premium pour completer les assets existants sans dupliquer les animaux deja presents.

# Scope

- Especes: mouton, chevre, poney.
- Stades: bebe, jeune, adulte.
- Poses: idle_1, idle_2, happy, sleeping, eating, celebrating.
- Format cible app: PNG transparents separes, gabarit 68x68, vue 3/4 topdown, pixel art cozy.

# Plan

1. Generer une spritesheet brute par espece en grille 3x6 sur fond #FF00FF.
2. Decouper chaque sheet en poses nommees.
3. Controler rapidement dimensions, transparence et coherence visuelle.
4. Documenter les prompts utilises et les sorties produites.

# Validation

- Les fichiers ne touchent pas les bords du cadre.
- Le fond magenta est transforme en transparence.
- Les noms de fichiers correspondent au pattern existant des compagnons.
