## What Was Built

**VaultManager.skipTask** (`lib/vault.ts`) — Nouvelle méthode dans `enqueueWrite` :
- Tâche récurrente avec date : avance via `nextOccurrence`, garde `- [ ]`, retire `✅`
- Tâche non-récurrente avec date : replace par `addDays(parseISO(date), 1)` (lendemain)
- Sans date : no-op (`return early`)

**useVault.skipTask** (`hooks/useVault.ts`) — Callback avec mise à jour optimiste identique au pattern `toggleTask` recurring. Exporté dans les deux useMemo (state object + dépendances).

**TaskCard** (`components/TaskCard.tsx`) — Prop `onSkip?: (task: Task) => void` ajoutée. `showSkip = !!onSkip && !!task.dueDate && !task.completed`. Bouton `⏭️ Passer` dans la zone `meta` avant les badges, avec haptic `selectionAsync`.

**Dashboard sections** — DashboardMenage, DashboardOverdue, DashboardVacation destructurent `handleTaskSkip` et le passent en `onSkip` aux TaskCard. `DashboardSectionWithTaskToggleProps` étendu avec `handleTaskSkip?` (optionnel).

**tasks.tsx + index.tsx** — `handleTaskSkip` callback sans aucun appel `completeTask`/`refreshGamification`. Toast `⏭️ Tâche reportée` ou `⏭️ Reportée à demain`.
