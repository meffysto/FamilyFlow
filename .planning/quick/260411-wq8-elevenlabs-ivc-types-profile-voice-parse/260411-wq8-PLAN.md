 * iOS Personal Voice (créée par l'utilisateur dans Réglages > Accessibilité > Voix personnelle)

2. **Import et fonction** :

```typescript
import * as Speech from 'expo-speech';

export async function getPersonalVoices(): Promise<Speech.Voice[]> {
  try {
    const allVoices = await Speech.getAvailableVoicesAsync();
    const highQualityVoices = allVoices.filter(voice =>
      voice.quality === Speech.VoiceQuality.Enhanced ||
      voice.quality === Speech.VoiceQuality.Premium
    );
    return highQualityVoices.sort((a, b) => a.name.localeCompare(b.name));
  } catch (error) {
    if (__DEV__) console.warn('[personal-voice] Erreur getAvailableVoicesAsync:', error);
    return [];
  }
}
```

Note sur `Speech.VoiceQuality` : expo-speech expose un enum `VoiceQuality` avec `Default | Enhanced | Premium`. Si l'enum n'est pas disponible à l'import (anciennes versions), fallback sur la comparaison string :

```typescript
voice.quality === 'Enhanced' || voice.quality === 'Premium'
```

Utiliser la comparaison string directe — plus robuste cross-version d'expo-speech et pas de dépendance au runtime de l'enum :

```typescript
const highQualityVoices = allVoices.filter(voice =>
  voice.quality === 'Enhanced' || voice.quality === 'Premium'
);
```
