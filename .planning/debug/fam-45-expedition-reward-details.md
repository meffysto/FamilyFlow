# FAM-45 — Détail des boosts dans les récompenses d'expédition

## Objectif

Rendre les récompenses d'expédition plus explicites pour les boosts temporels, notamment `boost_chance_doree` et `boost_production_2x`.

## Décision

Le détail est centralisé dans le moteur d'expédition pour être réutilisé dans le coffre de collecte et dans la liste des résultats.
