Comment ça marche

1

Créez votre espace famille

Ajoutez les parents, les enfants — l'app crée tout pour vous. Si vous avez déjà un dossier Obsidian, connectez-le directement.

Vivez votre quotidien

Cochez vos tâches, notez les biberons, planifiez les repas, suivez le budget. L'app vous suggère ce qui est important, rien ne se perd, rien ne s'oublie.

3

Récoltez les récompenses

Chaque tâche complétée rapporte des points. Débloquez des récompenses, maintenez votre série, et transformez le quotidien en jeu de famille.

FAQ

Questions fréquentes

FamilyFlow est-il gratuit ?+

Oui, entièrement gratuit et open source. Aucun abonnement, aucune limitation, aucune publicité.

Ai-je besoin d'Obsidian ?+

Non. FamilyFlow utilise le même format de fichiers (Markdown), mais vous n'avez pas besoin d'installer Obsidian. Si vous en avez déjà un, vous pouvez le connecter. Sinon, l'app crée tout pour vous.

Je pars de zéro, comment démarrer ?+

Au premier lancement, un assistant vous guide en 3 étapes : ajoutez les parents, les enfants, puis validez. FamilyFlow crée automatiquement un vault complet avec tous les dossiers et fichiers pré-remplis — tâches, courses, ménage, journal de bébé, rendez-vous. Vous êtes opérationnel en moins de 2 minutes.

Mes données sont-elles envoyées sur un serveur ?+

Non. Toutes vos données restent en local sur votre appareil. La synchronisation entre appareils passe par Obsidian Sync ou Syncthing, que vous contrôlez.

Comment fonctionnent les recettes ?+

Vos recettes sont stockées en texte simple (format Cooklang). L'app reconnaît automatiquement les ingrédients et les quantités pour générer votre liste de courses. Vous pouvez aussi importer des recettes depuis n'importe quel site web.

Comment fonctionne le suivi grossesse ?+

Ajoutez un profil avec le statut "grossesse" et la date de terme. L'app calcule les semaines d'aménorrhée, compare la taille du bébé à un fruit, et peut envoyer un récap hebdomadaire par Telegram. À la naissance, le profil bascule automatiquement en mode bébé.

Que font les suggestions intelligentes ?+

L'app analyse localement vos données (tâches en retard, RDV proches, stocks bas, courses manquantes...) et affiche des suggestions sur le tableau de bord. Optionnellement, vous pouvez activer l'IA (Claude) pour enrichir ces suggestions et poser des questions en langage naturel via la recherche globale. Toutes les requêtes sont anonymisées : les prénoms, lieux et données sensibles sont remplacés avant d'être envoyés.

Mon conjoint peut-il aussi utiliser l'app ?+

Oui ! Chaque parent a son propre profil. Pour partager les données entre deux téléphones, synchronisez le dossier via Obsidian Sync, Syncthing ou iCloud. Les deux parents voient les mêmes tâches, RDV et journaux en temps réel.

Sur quels appareils ça fonctionne ?+

L'app est disponible sur iOS (iPhone/iPad). Une version Android est prévue. Le dossier Markdown peut être édité sur n'importe quel appareil.
