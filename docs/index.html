Comment ca marche

1

Connectez votre vault

Pointez l'app vers votre dossier Obsidian. Vos fichiers Markdown sont votre base de donnees — aucun cloud proprietaire.

2

Organisez en un tap

Cochez vos taches, ajoutez des RDV, remplissez le journal bebe, planifiez vos repas. Tout est sauvegarde instantanement.

3

Gagnez des points

Chaque tache completee rapporte des points. Debloquez des loot boxes, maintenez votre serie, et rendez l'organisation amusante.

FAQ

Questions frequentes

FamilyFlow est-il gratuit ?+

Oui, entierement gratuit et open source. Aucun abonnement, aucune limitation, aucune publicite.

Ai-je besoin d'Obsidian ?+

FamilyFlow utilise un vault Obsidian comme source de donnees (fichiers Markdown). Vous pouvez utiliser Obsidian en parallele, mais ce n'est pas obligatoire.

Mes donnees sont-elles envoyees sur un serveur ?+

Non. Toutes vos donnees restent en local sur votre appareil. La synchronisation entre appareils passe par Obsidian Sync ou Syncthing, que vous controlez.

Qu'est-ce que le format Cooklang ?+

Cooklang est un format texte pour ecrire des recettes. Les ingredients sont marques avec @, les ustensiles avec #. FamilyFlow parse ces fichiers .cook pour extraire automatiquement les ingredients et generer vos listes de courses.

Comment fonctionne le suivi grossesse ?+

Ajoutez un profil avec le statut "grossesse" et la date de terme. L'app calcule les semaines d'amenorrhee, compare la taille du bebe a un fruit, et peut envoyer un recap hebdomadaire par Telegram. A la naissance, le profil bascule automatiquement en mode bebe.

Sur quels appareils ca fonctionne ?+

L'app est disponible sur iOS (iPhone/iPad). Une version Android est prevue. Le vault Markdown peut etre edite sur n'importe quel appareil via Obsidian.
