Refonte v4 — descriptive

Le changement
Plus de label abstrait ("Cycle", "Cadence"…). À la place, chaque ligne raconte ce qui change : "1 miel toutes les 7h37 → 7h00". Le label devient une phrase descriptive, le delta reste un badge isolé.

Pourquoi ça marche mieux

  • L'icône (⏱ / 📦) suffit comme repère visuel — pas besoin de mot pour la nommer
  • La phrase descriptive ("1 miel toutes les") contextualise la valeur — l'utilisateur sait ce qu'il regarde sans se demander
  • Les unités sont explicites (7h37, 5 miels) — pas de chiffre nu
  • Le badge delta isole l'amplitude du gain (−8 %, +1) sans encombrer la phrase
  • Layout en lignes (icône · description · transition) → relisible verticalement

Cohérence avec la barre de progression

  • La hero card dit "Prochain miel dans 3h12"
  • La carte upgrade dit "1 miel toutes les 7h37 → 7h00"
  • Même vocabulaire ("miel", durée formatée), pas de divergence
  • Quand toit endommagé : "Production en pause" au lieu d'un timer faux
