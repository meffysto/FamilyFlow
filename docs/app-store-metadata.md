## Sous-titre (30 caractères max)

## Description (FR)

Family Vault est l'app qui centralise toute la vie de famille dans un seul endroit — repas, tâches, rendez-vous médicaux, journal bébé, courses et bien plus.

Conçue pour les parents avec de jeunes enfants, Family Vault remplace les carnets, les post-it et les groupes WhatsApp par une app simple et complète que toute la famille peut utiliser.

VOTRE QUOTIDIEN, SIMPLIFIÉ
• Planifiez les repas de la semaine et générez la liste de courses
• Répartissez les tâches ménagères entre les membres de la famille
• Gérez le stock de la maison (frigo, placards, congélateur)
• Suivez le budget familial et scannez les tickets de caisse

SUIVEZ LA SANTÉ DE VOS ENFANTS
• Journal bébé complet : biberons, couches, sommeil, médicaments
• Rendez-vous médicaux avec dictaphone intégré pour les consultations
• Carnet de santé numérique avec courbes de croissance
• Suivi de grossesse semaine par semaine

TOUTE LA FAMILLE PARTICIPE
• Chaque membre a son profil avec son thème personnalisé
• Mode enfant sécurisé avec contrôle parental
• Système de points et récompenses pour motiver les enfants
• Défis familiaux et classement pour s'amuser ensemble

VOTRE VIE PRIVÉE PROTÉGÉE
• Toutes les données restent sur votre appareil
• Aucun compte à créer, aucune inscription
• Synchronisation via iCloud Drive ou Obsidian Sync
• Protection par Face ID / Touch ID / code PIN
• Compatible avec le vault Obsidian — vos données sont des fichiers Markdown lisibles

WIDGETS iOS
• Widget "Ma Journée" : repas, tâches et RDV du jour en un coup d'œil
• Widget "Journal Bébé" : dernier repas et stats du jour

ASSISTANT IA (OPTIONNEL)
• Posez des questions sur votre organisation familiale
• Suggestions intelligentes basées sur vos habitudes
• Fonctionne avec votre propre clé API — aucune donnée stockée sur nos serveurs

Family Vault est gratuite, sans publicité et sans abonnement.
