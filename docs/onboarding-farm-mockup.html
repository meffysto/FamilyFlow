Notes d'implémentation

Option A — Bande panoramique

Remplacement 1:1 de featureHeroEmojis. Un seul View avec fond dégradé + Image pour chaque sprite. Pas de layout change. Idéal si on veut aller vite.

Option B — 2 plans

Nécessite un container position:relative avec deux rangées absolute. Intègre les assets existants (poulailler, animaux, décos). La plus visuellement impactante.

Option C — Avant/Après

Deux colonnes flexbox séparées par un diviseur. Raconte une histoire de progression. Bonne cohérence avec le message "de la graine au jardin".
