Support

Family Vault — Aide et contact

Family Vault est une app gratuite, sans publicité et sans abonnement. Le support est assuré bénévolement.

Questions fréquentes

  • Où sont stockées mes données ? — Sur votre appareil, dans un vault Obsidian (fichiers Markdown). Rien n'est envoyé sur Internet sauf si vous activez l'IA optionnelle.
  • Comment synchroniser entre appareils ? — Placez votre vault dans iCloud Drive, ou utilisez Obsidian Sync.
  • J'ai perdu mes données — Vérifiez que votre vault est bien dans iCloud Drive (pas dans le container Obsidian). Vos fichiers .md sont lisibles avec n'importe quel éditeur de texte.
  • Face ID / le PIN ne fonctionne pas — Allez dans Réglages > Sécurité pour réinitialiser la protection.
  • Les widgets n'affichent rien — Ouvrez l'app une fois pour initialiser les données partagées avec les widgets.
